Plan de Estudio de Desarrollo Web

El siguiente programa organiza los principales de un curso introductorio de desarrollo web

Modulos del Curso

  • HTML
    • Estrucutura basica de documentos.
    • Etiquetas semanticas.
    • Enlaces y navegacion.
  • CSS
    • Selectores.
    • Modelos de caja.
    • Diseño responve.

Actividades

  1. Ejercicios Individuales
    1. Crear pagina personal.
    2. Construir un menu de navegacion.
  2. Proyecto Final
    1. Diseñar estructura del sitio.
    2. Implementar paginas principales.
    3. Publicar el proyecto.
